## 获取商品列表product_list ``` 目前是通过 access_token来获取商品Product的列表product_list。 access_token：访问开放平台的token; 接口地址 测试地址： http://openapi.weapp.weizzz.com/mall/products?access_token=xxxx 线上地址： http://openapi.weapp.weizoom.com/mall/products?access_token=xxxx ``` ## product_list详细举例说明 例子： ``` [{ "name": "短袖T恤", "display_price": 220.00, "sales": 26, "thumbnails_url": "http://chaozhi.weizoom.com/static/upload/20160921/1474450258043_377.jpg", "supplier": 2, "id": 11 }, { "name": "吊裆裤", "display_price": 399.00, "sales": 44, "thumbnails_url": "http://chaozhi.weizoom.com/static/upload/20160921/1474448967063_375.jpg", "supplier": 2, "id": 10 }] ``` ## product_list字段描述信息： ``` "items": "商品列表,数据结构为：[{},{},...]", [{ "name": "商品名称", "thumbnails_url": "商品缩略图的url地址", "display_price": "商品总库存", "id": "商品ID", "sales": "商品销量", "supplier": "供货商ID" }] ``` ## product_list文件下载 [product_list.json文件地址](http://dev.poseidon.com/static/wiki/product_list.json)
